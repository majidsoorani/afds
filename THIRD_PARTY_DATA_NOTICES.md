# Third-Party Data and API Notices

AFDS integrates with external data and API providers.
Use of those services is governed by each provider's terms.

## Important

- This repository does not grant rights to third-party datasets.
- Commercial users must obtain required provider licenses/subscriptions.
- Contributors must not commit restricted datasets or paid feed exports.

## Referenced Providers (non-exhaustive)

- OpenSanctions
- Have I Been Pwned (HIBP)
- MaxMind GeoLite/GeoIP data
- third-party vendor APIs
- Tor exit list and other threat-intel feeds

## User Responsibilities

- Review and comply with provider terms before production usage.
- Store provider API keys securely.
- Follow data retention and redistribution restrictions.
